# 🧠 React Quiz App

An interactive, timed multiple-choice quiz application built with **React** and powered by **Vite** for blazing-fast development and build performance.

---

## ⚡ Tech Stack

- [React](https://reactjs.org/) — UI library
- [Vite](https://vitejs.dev/) — Next-generation frontend build tool
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [JSON] — For question data
- [CSS] — Basic styling

---

## 🚀 Features

- ✅ Dynamic questions loaded from a JSON file
- ⏱️ 10-second countdown timer for each question
- 📊 Real-time scoring
- 🔁 Restart quiz at the end
- ⚡ Fast dev environment with Vite and hot module replacement (HMR)
